# Анализ использования смартфонов и продуктивности

Данный проект посвящён анализу датасета `Smartphone_Usage_Productivity_Dataset_50000.csv`, содержащего информацию о 50 000 респондентах: их возрасте, поле, роде занятий, времени использования телефона, сне, уровне стресса и потреблении кофеина. В ходе работы выполняются SQL-запросы к данным, загруженным в SQLite, и строятся визуализации с помощью Matplotlib/Seaborn.

Датасет содержит следующие поля (в коде используются выбранные столбцы):

- `Age` — возраст респондента
- `Gender` — пол (Male/Female/Other)
- `Occupation` — род занятий
- `Daily_Phone_Hours` — ежедневное время использования телефона (часы)
- `Sleep_Hours` — продолжительность сна (часы)
- `Stress_Level` — уровень стресса (1–10)
- `Caffeine_Intake_Cups` — количество выпитых чашек кофе в день
