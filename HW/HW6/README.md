# Сегментация клиентов торгового центра

## Описание проекта
Данный проект посвящен анализу данных клиентов торгового центра с целью их сегментации на основе демографических характеристик (возраст, годовой доход) и потребительского поведения (показатель расходов). Задача решается методами машинного обучения без учителя (кластеризация). Результатом является выделение устойчивых групп клиентов для дальнейшей маркетинговой стратегии.

## Датасет
*   **Источник:** [Mall_Customers.csv](https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python)
*   **Количество записей:** 200
*   **Признаки:**
    *   `CustomerID`: уникальный идентификатор клиента
    *   `Gender`: пол (Male/Female)
    *   `Age`: возраст (лет)
    *   `Annual Income (k$)`: годовой доход (в тысячах долларов)
    *   `Spending Score (1-100)`: показатель расходов (от 1 до 100), присвоенный торговым центром на основе поведения клиента.

## Структура проекта

*   `README.md`: данный файл с описанием проекта.

## Этапы работы
1.  **Загрузка и первичный анализ данных:** знакомство со структурой, типами данных, проверка на пропуски.
2.  **Предобработка:** кодирование категориального признака `Gender`, масштабирование числовых признаков с помощью `StandardScaler` для корректной работы алгоритмов.
3.  **Разведочный анализ данных (EDA):** построение гистограмм распределений, парных графиков и корреляционной матрицы для понимания взаимосвязей между признаками.
4.  **Выявление аномалий:** применение Isolation Forest для поиска выбросов, которые могут исказить результаты кластеризации.
5.  **Кластеризация и оценка:** применение пяти алгоритмов кластеризации:
    *   **KMeans** (с подбором оптимального числа кластеров по методу локтя, силуэту, индексам Дэвиса–Болдина и Калински–Харабаса).
    *   **Агломеративная кластеризация** (AgglomerativeClustering).
    *   **DBSCAN** (с подбором параметров eps и min_samples).
    *   **MeanShift** (с автоматическим определением ширины окна).
    *   **Affinity Propagation**.
6.  **Интерпретация результатов:** визуализация полученных кластеров и описание каждого сегмента на основе средних значений признаков.

## Результаты
*   Наилучший метод кластеризации – **KMeans с количеством кластеров k=6**.
*   Метрики качества для KMeans (k=6):
    *   Silhouette Score: 0.4284
    *   Davies-Bouldin Index: 0.8254
    *   Calinski-Harabasz Index: 135.1
*   Выделенные сегменты клиентов:
    1.  **Кластер 0:** Клиенты среднего возраста, доход выше среднего, высокий потребительский счет.
    2.  **Кластер 1:** Молодые клиенты, невысокий доход, но высокий счет (склонны тратить).
    3.  **Кластер 2:** Клиенты среднего/старшего возраста, доход ниже среднего, низкий счет (экономные).
    4.  **Кластер 3:** Молодые клиенты, доход выше среднего, очень высокий счет (самая привлекательная группа).
    5.  **Кластер 4:** Клиенты среднего возраста, доход выше среднего, умеренный счет (уравновешенные).
    6.  **Кластер 5:** Пожилые клиенты, высокий доход, низкий счет (состоятельные, но неактивные).

## Выводы
*   Сегментация клиентов позволила выделить 6 устойчивых групп, различающихся по демографическим и поведенческим признакам.
*   Наиболее перспективными для маркетинговых активностей являются кластеры 1 и 3 – молодые люди с высоким потребительским потенциалом.
*   Результаты кластеризации могут быть использованы для разработки таргетированных рекламных кампаний, программ лояльности и персонализации предложений.
