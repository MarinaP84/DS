README для домашнего задания №5

Сравнение регрессоров на датасете Medical Cost Personal

---

📋 Описание проекта

В данном проекте решается задача регрессии для предсказания индивидуальных медицинских расходов на основе демографических и поведенческих факторов. Датасет Medical Cost Personal содержит 1338 записей и 6 признаков.

Цель работы:

· Реализовать минимум 5 регрессоров из предложенного списка
· Провести гиперпараметрическую настройку для каждой модели
· Сравнить метрики и выбрать лучший регрессор для данного датасета

---

Характеристики датасета

Характеристика Значение
Источник Kaggle: Medical Cost Personal Dataset
Количество записей 1338
Целевая переменная charges — медицинские расходы (в долларах)
Признаки age, sex, bmi, children, smoker, region
Пропуски отсутствуют

Лучшая модель

XGBoost показал наилучший результат с R² = 0.8745 на тестовой выборке.

Преимущества XGBoost:

· Наивысшая точность предсказаний
· Эффективная работа с нелинейными зависимостями
· Встроенная регуляризация для предотвращения переобучения
· Хорошая скорость обучения

Лучшие параметры XGBoost:

```python
{
    'colsample_bytree': 0.8,
    'learning_rate': 0.1,
    'max_depth': 5,
    'n_estimators': 200,
    'subsample': 0.8
}
```

---

Детальный анализ моделей

1. XGBoost (R² = 0.8745)

· Самый высокий R²
· Хороший баланс скорости и качества
· Важность признаков: smoker > bmi > age

2. LightGBM (R² = 0.8721)

· Самая быстрая модель
· Результат близок к XGBoost
· Низкое потребление памяти

3. Gradient Boosting (R² = 0.8543)

· Хорошая точность
· Более медленное обучение

4. Random Forest (R² = 0.8512)

· Робастность к выбросам
· Отличная интерпретируемость (важность признаков)
· Чуть ниже точность

5. Ridge / Linear Regression (R² ≈ 0.78)

· Быстрое обучение
· Не улавливают нелинейные зависимости
· Значительно ниже точность
