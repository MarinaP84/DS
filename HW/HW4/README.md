Сравнение классификаторов для обнаружения мошеннических транзакций (Credit Card Fraud Detection)
Описание проекта
В данном проекте решается задача бинарной классификации для обнаружения мошеннических транзакций по кредитным картам. Датасет содержит 284 807 транзакций, из которых только 492 (0.17%) являются мошенническими – сильный дисбаланс классов.
Цель работы:
· Реализовать минимум 5 классификаторов из предложенного списка
· Провести гиперпараметрическую настройку для каждой модели
· Сравнить метрики и выбрать лучший классификатор для данного датасета
Используемые классификаторы
1. Gradient Boosting Classifier – градиентный бустинг
2. CatBoost Classifier – бустинг от Яндекса с автоматической обработкой категориальных признаков
3. XGBoost Classifier – экстремальный градиентный бустинг
4. KNeighbors Classifier – метод k ближайших соседей
5. Decision Tree Classifier – дерево решений
Результаты
Итоговые метрики:
Модель Precision (class 1) Recall (class 1) F1-score (class 1) Accuracy
Gradient Boosting 0.22 0.86 0.35 0.9943
CatBoost 0.69 0.82 0.75 0.9990
XGBoost 0.88 0.79 0.83 0.9994
KNN 0.92 0.74 0.82 0.9994
Decision Tree 0.70 0.80 0.74 0.9991
Лучшая модель
По результатам сравнения лучшей моделью выбран XGBoost с F1-score = 0.83 для класса мошеннических транзакций. Модель показывает оптимальный баланс между precision (0.88) и recall (0.79), что позволяет обнаруживать большинство мошенничеств с относительно низким уровнем ложных срабатываний.
Выводы
1. Для несбалансированных данных критически важно использовать стратификацию при разделении выборок и балансировку классов в моделях.
2. Ансамблевые методы (XGBoost, CatBoost) показывают лучшие результаты по сравнению с простыми моделями (KNN, дерево решений).
3. Gradient Boosting показал низкий F1-score из-за недостаточной балансировки (несмотря на использование весов, параметры подобраны неоптимально).
4. Гиперпараметрическая настройка позволяет значительно улучшить качество моделей – это видно на примере CatBoost и XGBoost по сравнению с Gradient Boosting.
5. Для задачи обнаружения мошенничества важнее всего высокий recall (чтобы не пропустить мошенничество), но с учётом precision, поэтому F1-score является лучшей метрикой для выбора модели.
